# Student Activity Tracker System

A full-stack web application inspired by Google Classroom with dynamic role assignment per class.

## Architecture
- **Frontend**: HTML/CSS/JavaScript (responsive design)
- **Backend**: Node.js with Express.js (RESTful API)
- **Database**: MySQL (relational database)

## Key Features
- Single account system with dynamic role assignment
- Class creation and management
- Activity tracking and submissions
- Member management with role updates
- Comprehensive reporting system

## Project Structure
```
Frontend/          # Client-side code
Backend/           # Server-side API
Database/          # SQL scripts and schema
Routes/            # API route definitions
Controllers/       # Business logic
Models/            # Data models
Views/             # HTML templates
Public/            # Static assets
```

## Setup Instructions
1. Install dependencies: `npm install`
2. Set up MySQL database using scripts in Database/
3. Configure environment variables
4. Run the application: `npm start`
